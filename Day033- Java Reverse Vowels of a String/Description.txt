Topic of the Day: Java Reverse Vowels of a String
Problem: A string s, reverse only all the vowels in the string and return it. The vowels are 'a', 'e', 'i', 'o', and 'u', and they can appear in both lower and upper cases, more than once.

Example 1:
Input: s = "hello"
Output: "holle"

Example 2:
Input: s = "leetcode"
Output: "leotcede"
 
Accepted
Runtime: 0 ms
Your input: "hello"
Output: "holle"
Expected: "holle"

Explanation:
Our task is to reverse only the vowels of string. One simple solution is to store all the vowels while scanning the string and placing the vowels in the reverse order in another iteration of string.
